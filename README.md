# 🔮 Futuristic Calculator

A sleek, glassmorphic web calculator with both **standard** and **scientific** modes — powered by a Spring Boot backend and Thymeleaf frontend.

> ⚡ Real-time results powered by server-side Java logic — no frontend math hacks!

---

## 🚀 Features

- ✨ Stylish glassmorphic UI (Standard & Scientific modes)
- ☀️ Light/Dark theme toggle
- 🧠 Smart backend calculation using Java `enum`, `Value`, and `Solution` classes
- 💡 Instant updates when entering expressions
- ✅ Fully functional with keyboard and mouse
- 🖥️ Built with **Spring Boot + Thymeleaf** — pure Java logic under the hood

---

## 🛠️ Tech Stack

- **Java 17**
- **Spring Boot**
- **Thymeleaf**
- **HTML5 / CSS3 / Vanilla JS**
- **Gradle**
